Project Introduction

The idea behind the project is to try and use ML to predict the future prices of Yu-Gi-Oh! cards. This is as much of a difficult task to do as it would be for predicting the future prices of anything. However, using game knowledge we can try to add features to the model that will help it see trends in cards based on how certain cards have done in the past/future.

At our disposal we have a dataset with every card in it from before mid-2020. I could update the algorithm I used to do this for a new website and give myself a better dataset. This first iteration can be found on Kaggle here.
